# AI Study Assistant Bot

A Telegram bot that quizzes you on your study materials.

## Features
- Upload PDF, PowerPoint or Word files
- AI generates 10 quiz questions from your content
- Explains wrong answers based on your file
- Built with Python, Claude AI, deployed on AWS

## Tech Stack
- Python
- python-telegram-bot
- Anthropic Claude API
- AWS (coming soon)
